# Smart Community Builder

## Overview
Smart Community Builder is a responsive web application that helps users discover nearby NGOs, healthcare centers, food banks, shelters, and other community support services.

## Features
- Search for nearby support services
- Browse categories like healthcare, food, and shelters
- Emergency assistance section
- Responsive user interface
- Modern design using Tailwind CSS

## Technologies Used
- HTML5
- Tailwind CSS
- JavaScript
- Python (AI server)
- Feather Icons

## Future Improvements
- User authentication
- Live database integration
- Google Maps API
- AI-powered recommendations
