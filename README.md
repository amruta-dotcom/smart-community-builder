## 📖 Overview

Smart Community Builder is designed to connect people with local support services quickly and easily. Whether someone needs to locate a nearby shelter, food bank, or NGO, the app aims to surface relevant help in a simple, accessible interface.

## ✨ Features

- 🔍 Search and browse nearby community support resources
- 🏠 Categorized listings (NGOs, food banks, shelters)
- 📱 Responsive, mobile-friendly design
- ⚡ Fast, lightweight front end
